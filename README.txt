Description:
	Simple application to retrieve and add vehicles 
	to a SQLServer DB with a normal Web frontend

*****************************/


1. Restore the vehicle.bak to a SQLServer instance
2. Default connectionstring authentication uses username:sa password:test. This can be changed in the app.config in the BussinessLayer
3. Open and run the application from VS (VS2012 was used to create the app)
4. Technologies used: C#, ASP.NET, .NET 4.0, javascript, jquery 1.10, AngularJS v1.2, CSS
5. The application only includes a search and add function, no updates or delete.
6. Functionaltiy showcased: 
	AJAX using C# backend with SQLServer and aspx web frontend 
	simple AngularJS usage 
	simple UI effects using jquery 
	javascript closures
